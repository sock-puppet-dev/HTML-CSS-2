HTML and CSS: The Comprehensive Guide — Jürgen Wolf

HTML-файл сам по себе — это обычный текстовый файл.

<h1>Hello</h1>
<p>This is a paragraph</p>

Но браузер читает этот текст и превращает его во внутреннюю структуру, с которой уже можно работать.
Эта структура называется DOM (Document Object Model) (Объектная Модель Документа)
То есть браузер берёт HTML-код и строит из него дерево объектов.

HTML-файл → браузер читает → создаёт DOM → показывает страницу

Каждый объект в DOM-дереве называется node.
Например, в DOM узлами являются:
<html>
<head>
<title>
<body>
<h1>
<p>

Зачем нужен DOM, DOM нужен, чтобы JavaScript мог менять страницу.

HTML — это исходный текст страницы.
DOM — это объектная структура, созданная браузером из HTML.
Node — это любой узел этой структуры.
JavaScript может управлять DOM.
